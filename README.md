# KnowledgeTree

研究上の問い、メモ、文献を有向グラフとして整理する Windows 向けデスクトップアプリケーションです。PyQt6 で動作します。

## 動作環境

- Windows
- Python 3.11

## セットアップ

プロジェクトのルートディレクトリで、仮想環境を作成して依存パッケージを導入します。

```powershell
py -3.11 -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
```

## 起動

通常は、プロジェクト直下の `run.bat` をダブルクリックしてください。仮想環境の `pythonw.exe` を使うため、アプリ実行中にターミナルは表示されません。

ターミナルから起動し、エラーを確認したい場合は次を実行します。

```powershell
.\.venv\Scripts\Activate.ps1
python main.py
```

初回起動時は「プロジェクトを開く」画面が表示されます。ここから既存プロジェクトを開くか、`新規作成` を選んで開始してください。新規プロジェクトには操作確認用のデモデータが作成されます。

## 基本操作

### プロジェクト

- `ファイル > プロジェクトを開く…` でプロジェクト一覧を開きます。
- 一覧から `新規作成`、`開く`、`削除` を行えます。開いているプロジェクトは削除できません。
- プロジェクトのデータは `userdata/projects/<プロジェクト名>/` に保存されます。バックアップするときは、このプロジェクトフォルダ全体をコピーしてください。

### 保存

グラフとプロジェクト設定は手動保存です。

- `Ctrl+S` または `ファイル > 保存` で保存します。
- 未保存の変更がある間は、ウィンドウタイトルの先頭に `*` が付きます。
- 未保存のまま閉じる場合は `Discard / Cancel` が表示されます。`Discard` はグラフとプロジェクト設定の未保存変更を破棄します。

全体設定と文献マスタは、それぞれのダイアログ内にある `Save / Cancel` で確定します。文献マスタを保存した後、文献ノードの表示が更新された場合は、グラフ側も `Ctrl+S` で保存してください。

### キャンバスの移動と表示

| 操作 | 動作 |
| --- | --- |
| 背景をドラッグ | キャンバスを移動 |
| ホイール | 垂直スクロール |
| Shift + ホイール | 水平スクロール |
| Ctrl + ホイール | 拡大・縮小 |
| `F` | グラフ全体を表示 |

### ノード

ノードには次の 3 種類があります。

- **問いノード**：問いを表します。タイトル、本文、子の条件（なし／AND／OR）を編集できます。
- **メモノード**：アイデア、補足、成果などを自由に記録します。
- **文献ノード**：文献マスタ内の論文・書籍・Webサイトを参照します。

背景を右クリックすると、問い・メモ・文献ノードを追加できます。ノードまたはエッジをダブルクリックすると、右側のインスペクタを表示します。

複数ノードを選択してまとめて動かすこともできます。

1. ノードを 1 つクリックして選択します。
2. Shift を押しながら別のノードをクリックして追加選択します。
3. 選択中のいずれかのノードをドラッグすると、全選択ノードが同時に移動します。

複数選択中はインスペクタを編集できません。エッジは常に単一選択です。

### エッジ

- ノードの外縁を別ノードへドラッグすると、有向エッジを追加します。
- ノード外縁から背景へドラッグして離すと、新しい問いノードを作り、そこへ接続します。
- エッジの端をドラッグすると、接続先または接続元を別ノードへ付け替えられます。背景で離すとエッジを削除します。
- エッジを右クリックすると、そのエッジ上へノードを挿入できます。
- エッジラベルはドラッグして表示位置を調整できます。

関係の種類は、始点・終点のノード種類に応じて自動選択されます。新規プロジェクトの既定値は次の 3 種類です。

| ラベル | 接続 | 用途 |
| --- | --- | --- |
| `refines` | 問い → 問い | 問いを具体化・分解する |
| `contributesTo` | 文献・メモ → 問い | 問いへの理解や回答を前進させる |
| `leadsTo` | 文献 → 文献 | 以前の成果から後の成果への発展を表す |

### 設定と文献管理

- ツールバーの歯車から **プロジェクト設定** を開けます。関係種類、接続可能なノード種類、関係色、ノード種類ごとの色を編集できます。
- 本のアイコン、または `ファイル > 文献を管理…` から **文献を管理** を開けます。
- 文献管理では Paper、Book、Website をタブごとに一覧・編集できます。CSV は Excel で開ける Shift-JIS（CP932）形式で保存されます。
- `ファイル > 全体設定…` では、次回起動時に最後のプロジェクトを開くかを設定できます。
